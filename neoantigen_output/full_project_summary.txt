DEK-AFF2 Fusion Neoantigen Discovery — Full Project Summary
============================================================
Date: 2026-03-22
Source: Hartwig Medical Foundation WGS
Diagnosis: Stage IV NSCLC with neuroendocrine differentiation

============================================================
WORKING MODEL
============================================================

This summary describes a computational screening exercise around a
reported DEK-AFF2 fusion (Hartwig/LINX: DEK exon 7 -> AFF2 exon 9).
It should be treated as hypothesis-generating, not clinically validated.

Using canonical DEK and AFF2 transcripts, the current in-frame working
model places the protein junction at:

  ...ESEEE|AVEKA...
      DEK   AFF2

Under this model:
- DEK contribution ends at aa 254
- AFF2 contribution starts at aa 454
- the fusion protein is 1112 aa long

This sequence model is internally consistent, but it is still an
inference until the exact LINX breakpoint coordinates and/or RNA fusion
evidence are checked.

============================================================
WHAT WAS GENERATED
============================================================

Peptides spanning the modeled junction:
- 34 Class I peptides (8-11mers)
- 14 Class II peptides (15mers)

Examples from the current peptide set:
- Class I: ESEEEAVEK, SEEEAVEKA, KESEEEAVEK, KESEEEAVEKA, EAVEKAKPR
- Class II: DKESEEEAVEKAKPR, KESEEEAVEKAKPRN, ESEEEAVEKAKPRNN

All generated peptides cross the modeled DEK|AFF2 boundary and therefore
do not exist as contiguous sequences in the native proteins.

============================================================
AGRETOPICITY / DAI
============================================================

DAI here is used only as a rough ranking heuristic:

  DAI = IC50_wildtype / IC50_fusion

The current output compares each fusion peptide against both its DEK-side
and AFF2-side wildtype counterparts, then uses the lower available DAI as
the conservative summary.

Examples from the current conservative ranking:

1. KESEEEAV on HLA-B*40:02
   - Fusion IC50: 95.7 nM
   - Conservative DAI: 103.4

2. ESEEEAVEKAK on HLA-A*68:01
   - Fusion IC50: 176.7 nM
   - Conservative DAI: 54.5

3. EAVEKAKPR on HLA-A*68:01
   - Fusion IC50: 30.6 nM
   - Conservative DAI: 14.1

4. KESEEEAVEKA on HLA-B*40:02
   - Fusion IC50: 70.7 nM
   - Conservative DAI: 3.6

5. SEEEAVEKA on HLA-B*40:02
   - Fusion IC50: 65.3 nM
   - Conservative DAI: 1.4

Interpretation:
- Some candidates remain interesting even under conservative aggregation.
- Others look much less distinctive once both WT sides are considered.
- DAI still does not model antigen processing, abundance, clonality, or
  T-cell receptor recognition.

============================================================
WHAT IS MOST LIKELY CORRECT VS STILL UNCERTAIN
============================================================

Likely correct now:
- The Class I reports are referencing the 34 modeled Class I peptides only.
- The reported strong/elite binders match the sequences in
  `junction_peptides.fasta`.
- The DAI tables now preserve both WT comparisons rather than keeping only
  the most favorable side.

Still uncertain:
- Whether the modeled junction sequence is the patient's true fusion
  peptide sequence.
- Whether a different breakpoint or transcript would produce a different
  junction, including different candidate peptides.
- Whether any predicted binders are actually presented by the tumor.
